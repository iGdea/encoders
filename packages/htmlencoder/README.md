htmlyer
======

[![NPM Version][npm-image]][npm-url]
[![NPM Downloads][downloads-image]][npm-url]
[![NPM License][license-image]][npm-url]

Html encoders
# Install
```
npm install htmlyer --save
```

# Usage

```javascript
const {
    htmlEncode,
    jsEncode,
    urlEncode,
    jsonStringify,
    input2html,
} = require('htmlyer');

```
